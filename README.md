### Description

Just another text editor, aka Jate, is a code editing PWA for javascript.

### Usage
As a progressive web application, Jate is responsive, installable, and can be used offline for any text editing needs. 

### Technologies Used

- Javascript
- Node JS
- Webpack
- IndexedDB/idb
- Babel
- Workbox
